# Contributing to Smart Cleaner

Thank you for considering contributing! This document outlines the basic guidelines.

## How to Contribute

1. Fork the repository.
2. Create a feature branch from `main`.
3. Make your changes, following the coding style defined in `.editorconfig`.
4. Test your changes by running the application (`npm start`).
5. Commit with a clear, descriptive message.
6. Push your branch and open a pull request.

## Code Style

- Use 2 spaces for indentation (see `.editorconfig`).
- Follow modern JavaScript (ES2020+) conventions.
- Keep functions small and focused.
- Use `const` and `let` – avoid `var`.

## Commit Messages

- Use the present tense ("Add feature" not "Added feature").
- Limit the first line to 72 characters.
- Reference issues or pull requests when applicable.

## Reporting Issues

Open an issue with a clear title and description. Include steps to reproduce if it's a bug.
